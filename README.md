Porting MIUI to HTC_OneX


1. clone this repo
---------------
    git clone  https://github.com/trulyliu/HTC_OneXT_MIUI.git

2. Put MIUI_OneX rom in miui_OneX dir
---------------

3. pack your own MIUI for HTC OneXT
---------------
    cd HTC_OneXT_MIUI
    ./goPack
